# Kernel-SPD-Pooling
The code of PR 2019 paper "Learning a robust representation via a deep network on symmetric positive definite manifolds"

An example is put in 
```
examples/minc
```
you can run it by
```
bash examples/minc/minc_conv_all.sh
```
